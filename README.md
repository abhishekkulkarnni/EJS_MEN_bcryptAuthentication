
# _User Authentication WebApp By Using 'bcrypt' Library_

## _Technologies Use :_
##### EJS(Embedded javaScript), Node.js, Express & MongoDB

## _How To Run :_
``` javascript
nodemon app.js
```

## _Functionality :_
##### Use bcrypt library to hash passwords for strong user authentication.
